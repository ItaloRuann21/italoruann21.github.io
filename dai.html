HTML - Estrutura do documento

Container tag - Delimitam partes do documento
<!--...--!> - Colocar comentário
 <!DOCTYPE> - informar ao navegador qual é a versão do HTML.(Navegadores ja suportam todas as versões)
<head> </head> - Cabeçalho do documento
<title> </title> - Tag de título do documento
<html> </html> - Início e fim do documento
<body> </body> - Onde o site vai ser feito.
<lang="pt-br"> - Definir a linguagem do documento para o navegador


HTML - Elementos básicos

<h1,h2,h3,h4,h5,h6> - Tags de título. Quanto maior o número, menor o texto.
text-align="center" - Alinha textos podendo ser: center, left, right, justify
valign - Alinhamento vertical.(top, buttom, middle, baseline)
<p> </p> - Parágrafos
<br /> - Quebra de linha
nbsp; - Serve para separar duas palavras uma em baixo da outra
<hr /> - Dividir texto ou objeto em linha horizontal
width - Largura. Pode ser em pixels (px) ou percentual (%) size - Tamanho da letra, fonte ou algum objetivo. Pode ser em pixels ou percentual.
<img /> - Tag para inserir uma imagem.
src - Insere o src dentro da tag img. O src é o link ou diretório de algo.
alt- Coloca depois do src. serve para colocar uma descrição na imagem.
height - Altura da imagem.
<b> <b/> - Negrito no texto.
<i> </i> - Itálico no texto.
<u> <u/> - Sublinhar texto.
<sup> </sup> - Sobescrever texto.
<sub> </sub> - Subscrever o texto. <strong> </strong> - Negrito no texto e o mais indicado pela W3C.
<small> </small> - Texto pequeno.
<tt> </tt> - Deixa texto com espaçamento regular.
<ol> </ol> - Lista ordenada. <ul> </ul> - Lista não ordenada.
<li> </li> - Item de cada lista.
type="1" - Números arábicos (1,2,3,4...)
type="a" - Letras minúsculas.
type="A" - Letras maiúsculas.
type="i" - Números romanos em minúsculos.
type="I" - Números romanos em maiúsculos.
<dl> </dl> - Lista de definições. é quase igual a tag ol e ul.
<dt> </dt> e <dd> </dd> - Cada item é adicionado através dessas tags.
Exemplo Abaixo:


bgcolor="cor" - Cor de fundo
background="imagem" - Imagem de fundo
<a> </a> - Tag usada para criar links
href="link" - Destino do link
target="valor" - Define o valor de um link. Se ele pode ir direto no link ou abrindo novas abas e etc.
_blank - É um valor do target que abre uma nova janela.
_self - Valor do target que abre na própria janela.
<a href="#ancora">texto< - Acessar um local específico da página automaticamente e rápido.
<a name="ancora">texto< - Onde o local vai ser especificado através do #ancora.
<table> </table> - Tag para criar tabelas.
<caption> </caption> - Título de uma tabela.
<tr> </tr> - Tag de linha.
<td> </td> - Célula de dados.
<th> </th> - Célula de título.
Cellpadding - Distância da borda/contexto.
Cellspacing - Distância entre células.
bordercolor - Cor da borda.
bordercolorlight - bordas claras com visualização tridimensional.
bordercolordark - bordas escuro com visualização tridimensional.
Colspan - Mesclar colunas.
Rowspan - Mesclar linhas.
<table border="2" "cellpadding"10"> - Define o espaçamento entre as bordas e o conteúdo do texto.
<table border="2" "Cellspacing"10"> - Deefine o espaço entre as células da tabela.

HTML5 e Formulários

Action - Nome do arquivo que receberá os dados do formulário.
Method - Forma de envio de dados se é GET ou POST.
<form> </form> - Todo formulário deve, sem exceção, possuir essa tag de abertura e fechamento!
O formulário pode conter: text fieds, checkboxes, radio-buttons, submit buttons e etc.
Exemplo abaixo:
<form action="gravar_dados.php" method="post">
//Campos do Formulários
</form>
<input> - Criar vários tipos de campos alterando atributos type
type - Informa o tipo de campo.
name - Recebe os dados do formulário.
size - Define o tamanho do campo da caixa de texto que será exibido pelo navegador,
maxlenght - Define a quantidade máxima de caracteres permitidas no campo de texto.
value - Utilizado para pré-definir um texto para o campo.
Tipos de type text, password, submit, reset, radio, checkbox
<textarea name="obs" cols="20 rows="5"></textarea> - Serve para colocar um campo grande de pergunta ou resposta.
<label> - Representa uma legenda para um item em interface do usuário.
<fieldset> - É usado para agrupar elementos, assim como os label, dentro de um formulário web.
<legend> - Representa um rótulo para o conteudo do seu ancestral (fieldset).
<footer> - Especifica o rodapé de um documento ou seção. Normalmente o rodapé de um documento contém o autor do documento, informações de copyright, links para termos de uso e informações de contato.
<nav> - Representa uma seção da página que contém um conjuntos de links de navegação.Geralmente esse elemento é utilizado no menu principal e nos blocos de links colocados no rodapé.
<aside> - Representa um conteúdo secundário que não faz parte do conteúdo principal do site. Esta marcação é bastante utilizada em sidebars e publicidade.
<section> - Delimita as seções de um documento.
<article> - Delimita um conteúdo que se constitui em um componente autossuficiente que pode ser distribuído ou reusado de forma independente. É recomendado o uso deste elemento em posts de fóruns e blogs, notícias e comentários.
<main> - Define o conteúdo principal da página, ou seja, representa o conteúdo mais importante da página.
tel - Campo usado para armazenar um telefone. Por não existir um padrão definido para números de telefone, este tipo não possui máscara de formatação ou validação. Se necessário, a validação deve ser feita com o uso do atributo pattern.
Exemplo abaixo>
<input type="tel" name="tel"
Email - Campo de email com formatação e validaçao.
Exemplo abaixo:
<input type="email" name="email">
Url - Campo para endereço url com formatação e validação.
Exemplo abaixo:
<input type="url" name="site">
Datetime - Campo destinado a receber data e hora.
Exemplo abaixo:
<input type="datetime" name="data">
datetime-local - Campo destinado a receber data e hora segundo a hora local, ou seja trata automaticamente a diferença entre fusos horários.
Exemplo abaixo>
<input type="datetime-local" name="data">
date - Campo destinado a data.
Exemplo abaixo:
<input type="date" name="data">
month - Campo destinado a receber mês e ano.
week - Campo destinado a receber a semana.
time - Campo destinado a recever hora.
number - Campo destinado a entrada de números.
range - Campo destinado a receber um número de um intervalo.
Exemplo abaixo:
<input type="range" name="volume" min="0" max="100">
color - Campo seletor de cor
Exemplo abaixo: <input type="color" name="cor">
autofocus - Atribui o foco ao campo com o atributo quando a página é carregada.
placeholder - Atribui dentro de um input um texto que será apagado quando o usuário começar a digitar dentro de um campo.
required - Determina que o preenchimento do campo é obrigatório.
maxlenght - Limita a quantidade de caracteres em um campo.
pattern - Determina que o preenchimento do campo deve seguir um padrão definido por uma expressão regular.
audio - Adiciona um som na página. É necessário declarar o caminho do arquivo de som no atributo src do elemento. O atributo control adiciona controles de audio como os botões play e pause e controle de volume.
Exemplo abaixo:
<audio controls>
<source src="kalimba.mp3" type="audio/ mpeg">
Seu navegador não suporte o elemento de audio.
</audio>
video - Adiciona um elemento de vídeo na página. Da mesma forma que no elemento de audio, é necessário declarar o caminho do arquivo. O atributo control adiciona os controles de vídeo como play, pause e
Exemplo abaixo:
<video width="320" height="240" controls>
<source src="amostra.mp4" type="video/mp4">
Seu navegador não suporte o elemento de video.
</video>
